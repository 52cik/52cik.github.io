我的 GitHub 100 连击

发表于 2016-05-05 | 分类于 狂人日妓 | 评论数：

终于达成 gayhub 的第一个100连击了，感觉自己整个人颜色都不一样了，完全蜕变了。

PS: GitHub 汉化插件 52cik/github-hans 感兴趣的赶紧 get 起来吧。

遇到瓶颈

在这之前，我感觉自己遇到瓶颈了，感觉目前的东西怎么看都不顺眼，想重写，怎么都写不出，就像是便秘一样。记得我第一次遇到瓶颈的时候，有大神推荐我看书，我看了些基础书以及进阶书籍，就是网上推荐的前端读书路线那种。还真别说，不到一个月，就明显突破了，知道自己要做什么，该做什么了。可是这次不行，我也看来各种书，但是然并卵，不知道为什么，也许是我沉淀的不够吧，反正这次没办法靠读书突破。

GitHub 连击计划

看了 phodal 大神的神奇经历后，突然知道我该做什么了，于是我就开始了我的第一个100连击计划。从过年回来后，到现在，终于100天了。从一开始的信心满满，到为了提交而提交，到顿悟，到什么都想参与，感觉自己发生了质变。

三个月的蜕变

第一个月

第一个月基本就是在写博客，然后是每天更新一部分汉化插件，顺便也做了两个小东西。
其实在第20天开始，感觉自己完全是为了提交而提交，而不是自主想做东西，但是到了40多天就完全变了，感觉什么都想做了。

第二个月

30天左右，已经到了什么非常疲倦的状态无奈找了几个简单的开源项目参与了下，顺便造个轮子。

第三个月

可是第三个月，就开始什么都想做一做了，不过基本都是造轮子。
就算是造轮子，那也比什么都不做的好，至少学会了各种轮子的生产过程以及运行原理。

500 连击目标

phodal 大神，目前连击576天了，我不知道能不能坚持到500天，也许因为特殊情况，哪天不能提交的话，只能重新开始，人生能有几个500天？
不过无关乎数字，我们要养成这样的习惯，每天都参与下开源，或者每天都逼自己写几行代码，这样才能逼自己变强。

后序

纸上得来终觉浅，绝知此事要躬行。 –– 陆游
实践是检验真理的唯一标准。 –– 毛主席

虽然他们都不是程序员，但是说的很有道理。 (●′ω`●)
